 Web Log Analysis with PySpark

Этот проект демонстрирует, как обрабатывать и анализировать веб-серверные логи (например, Apache) с использованием **PySpark**. Он включает:

- Парсинг логов с использованием регулярных выражений
- Преобразование в структурированный DataFrame
- Аналитика: топ IP, распределение HTTP-кодов, активность по часам
- Визуализация результатов
